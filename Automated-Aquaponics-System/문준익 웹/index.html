프로젝트 목표

“현장에서 돌아가는 자동화 스마트팜”을 최종 목표로, 단계별로 확장 가능한 구조를 먼저 설계합니다.

1) 통합 관제

3개의 SBC + 안전허브를 PC에서 하나로 모니터링

  • TCP/IP 기반 상태 수집
  • 카메라(웹캠/뎁스) 스트림 관리

2) 실시간 센서

수경·양식장 센서허브(STM32)가 센서값을 묶어서 전송

  • pH / TDS / 온습도 / 수온
  • 수위는 안전허브에서 독립 처리

3) 이상 감지 & 알람

임계값 기반 이상 상태 표시 → 추후 알림 연동

  • 화면 경고 배너
  • 이벤트 로그
